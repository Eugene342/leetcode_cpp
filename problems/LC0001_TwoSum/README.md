# LC0001 — Two Sum

LeetCode: https://leetcode.com/problems/two-sum/

Difficulty: Easy  
Tags: array, hash-table

Кратко:
Дан массив целых чисел `nums` и целевое значение `target`. Нужно найти индексы двух чисел в массиве, сумма которых равна `target`. Гарантируется, что существует ровно один ответ, нельзя использовать один и тот же элемент дважды.

Подход:
- Однопроходный хеш‑контейнер (unordered_map).
- Для каждого элемента `nums[i]` вычисляем `need = target - nums[i]` и проверяем, встречалось ли ранее `need` — если да, возвращаем индексы.
- Сложность: время O(n), память O(n).

Граничные случаи:
- Длина < 2 — по условиям LeetCode не случается.
- Повторяющиеся значения корректно обрабатываются за счёт хранения индексов.

Пример (self-test в `two_sum.cpp`):
- Вход: nums = [2,7,11,15], target = 9
- Ожидаемый выход: индексы 0 и 1 (порядок не важен)

Интеграция:
- Файл `two_sum.cpp` содержит `main()`, который выводит `OK` и возвращает 0 при успешном самотесте — удобно для `ctest`.
- Запуск локально (в корне репозитория):
  mkdir -p build && cd build
  cmake .. -DCMAKE_BUILD_TYPE=Release
  cmake --build . --parallel
  ctest --output-on-failure
