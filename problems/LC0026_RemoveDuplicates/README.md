# LC0026 — Remove Duplicates from Sorted Array

LeetCode: https://leetcode.com/problems/remove-duplicates-from-sorted-array/

Difficulty: Easy  
Tags: array, two-pointers

Кратко:
Дан отсортированный по неубыванию массив `nums`. Нужно удалить дубликаты на месте так, чтобы первые `k` элементов массива были уникальными значениями, и вернуть `k` (количество уникальных элементов). Отношение порядка элементов среди уникальных сохраняется.

Подход:
- Метод двух указателей (slow / fast).
- `slow` указывает на позицию последнего уникального элемента; `fast` пробегает массив. Когда `nums[fast] != nums[slow]`, продвигаем `slow` и записываем `nums[fast]` в `nums[slow]`.
- Сложность: время O(n), память O(1).

Граничные случаи:
- Пустой массив -> вернуть 0.
- Все элементы одинаковы -> вернуть 1.

Пример (self-test в `remove_duplicates.cpp`):
- Вход: nums = [0,0,1,1,1,2,2,3,3,4]
- После выполнения первые k = 5 элементов должны быть [0,1,2,3,4]
- `main()` в `remove_duplicates.cpp` проверяет корректность и возвращает 0 при успехе.

Интеграция:
- `remove_duplicates.cpp` содержит `main()` и возвращает 0 в случае успешной проверки — подходит для `ctest`.
- Сборка/тесты как в корневом README (cmake + ctest).
