# LC0011 — Container With Most Water

LeetCode: https://leetcode.com/problems/container-with-most-water/

Сложность: Medium  
Теги: array, two-pointers, greedy

Краткое описание:
Дан массив высот вертикальных линий. Нужно выбрать две линии, которые вместе с осью X образуют контейнер, в который можно поместить максимальное количество воды. Вернуть максимальную площадь.

Идея решения:
- Используем два указателя: left = 0 и right = n-1.
- Вычисляем площадь между текущими указателями: (right - left) * min(height[left], height[right]).
- Чтобы потенциально увеличить площадь, двигаем тот указатель, у которого меньшая высота.
- Повторяем, пока left < right.

Сложность:
- Время: O(n)
- Память: O(1) дополнительной

Примеры (self-test в `max_area.cpp`):
- [1,8,6,2,5,4,8,3,7] -> 49
- [4,3,2,1,4] -> 16

Примечание:
- В `max_area.cpp` есть `main()` с самопроверками: он выводит `OK` и возвращает 0 при успешных тестах — удобно для `ctest`/CI.
