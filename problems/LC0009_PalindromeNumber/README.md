# LC0009 — Palindrome Number

LeetCode: https://leetcode.com/problems/palindrome-number/

Сложность: Easy  
Теги: math, implementation

Кратко:
Дано целое число `x`. Нужно определить, является ли его десятичная запись палиндромом.

Безопасный и эффективный подход (в этом репозитории):
- Реверсируем только вторую половину числа (`rev_half`) и сравниваем с первой половиной (`x`).
- При чётной длине цифр: сравниваем `x == rev_half`.
- При нечётной длине: средняя цифра принадлежит либо первой, либо второй половине — отбросив среднюю цифру из `rev_half` (делением на 10) получаем сравнение `x == rev_half / 10`.
- Обрабатываем заранее отрицательные числа (false) и числа, заканчивающиеся на 0 (false, кроме 0).

Преимущества:
- Нет риска переполнения при реверсе полного числа.
- O(1) дополнительной памяти.
- Время O(d) — d число цифр (в int <= 10).

Примеры:
- 121 -> true
- -121 -> false
- 10 -> false
- 0 -> true

Файл `palindrome_number_half.cpp` содержит реализацию и `main()` с тестами.
Запуск через CMake/ctest: программа должна возвращать 0 и выводить `OK` при успешных тестах.
